## Installation
`composer require elliot-sawyer/silverstripe-typesense`.

For development or forks, add this to your composer.json

```json
    "repositories": {
        "elliot-sawyer/silverstripe-typesense": {
            "type": "vcs",
            "url": "git@codeberg.org:0x/silverstripe-typesense.git"
        }
    },
```
